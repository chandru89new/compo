Buttons

Buttons can be created by using the button tag or by adding class="button" to the a tag.

For additional styling:
<a class="button buttonPrimary">

<a> button buttonPrimary buttonDisabled buttonClicked

Toggles

Toggles are basically
<input type="checkbox">
with a class.

The default toggle is
class="toggleSimple"
and the other toggle is
class="toggleAlt"

Add toggleDual to each input class and you can use the toggle for switch options (instead of true and false).
